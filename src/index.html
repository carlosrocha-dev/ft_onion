Bem-vindo ao CPF Hidden Service!

Este é um serviço oculto acessível através da rede Tor.

O que é Tor?
Tor (The Onion Router) é um software livre e uma rede aberta que ajuda você a se defender contra uma forma de vigilância de rede que ameaça a liberdade e a privacidade pessoal, atividades comerciais confidenciais e relacionamentos, além da segurança do Estado.

O que é um serviço Onion?
Um serviço Onion é um servidor web que está disponível através da rede Tor, utilizando um domínio especial que termina em ".onion". Ele proporciona anonimato tanto para o servidor quanto para os usuários que acessam o serviço.

O que é Deep Web?
Deep Web refere-se a partes da internet que não são indexadas pelos mecanismos de busca padrão. A Deep Web inclui, entre outros, serviços Onion acessíveis através da rede Tor.
